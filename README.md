# ⚡ Net-Wraith Flasher

Control. Flash. Dominate.

A web-based flasher built for ESP8266 devices, designed to deploy multiple Net-Wraith firmwares directly from the browser using the Web Serial API.

---

## 🧠 Overview

**Net-Wraith Flasher** is a lightweight, browser-powered flashing tool that lets you install firmware on ESP8266 devices without downloading external tools.

Built for speed, simplicity, and control — plug in your device, click flash, and you're done.

---

## 🚀 Features

* ⚡ **One-click flashing** via browser
* 🔌 **No drivers / no installs** (Web Serial API)
* 🧩 **Multiple firmware support**
* 📡 **ESP8266 optimized**
* 🔄 **OTA-ready ecosystem**
* 🎯 Clean cyberpunk UI

## 🖥️ How It Works

1. Connect your ESP8266 via USB
2. Open the flasher website
3. Select firmware
4. Click **Connect & Flash**
5. Done — firmware is installed directly over USB
